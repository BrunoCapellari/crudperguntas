Um CRUD para criar perguntas onde elas serão salvas em um arquivo JSON

Para executa-lo, abra o arquivo "main.py" e inicie o código.

Para adicionar uma questão, preencha o campo "Pergunta" com sua respectiva pergunta, e coloque as alternativas nos campos "Opção 0","1" e "2.
E para especificar qual alternativa está correta, simplesmente digite o número respectivo da opção no campo "Alternativa Correta". Clique no botão 
"Adicionar Pergunta" e a pergunta ira ser adicionada no arquivo .JSON

Caso deseje atualizar uma pergunta, clique nela na lista abaixo dos botões e ela ira ser aberta a modificações nos campos. Faça quaisquer alterações e finalize
apertando o botão "Atualizar Pergunta"

Para remover uma pergunta, novamente selecione ela na lista e aperte o botão "Remover Pergunta"

Quando estiver adicionando multiplas perguntas, utilize o botão "Limpar" para que os campos sejam reiniciados.

E quando desejar sair, aperte o botão "Sair"
